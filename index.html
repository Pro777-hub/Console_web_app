What is Visual Studio?

Visual Studio is a free coding platform mainly aimed at creating webpages. It is quite more advanced than brackets and codepen io. Similar to Brackets,You can add extensions to it. Visual Studio focuses a lot on javascript.

Console

Console screen is used to detect errors in your javascript code like syntax errors. It is included in the option "Inspect". To access it,Right click and select inspect OR use the keyboard shortcut Ctrl + Shift + I,then select console from the screens shown.
